## 1. 메인 대시보드 (`/dashboard`)

### ONEWMS 위젯 표시
- [ ] **"ONEWMS 연동 상태"** 카드가 표시되는가?
- [ ] **실패 주문** 통계가 표시되는가?
  - [ ] 실패 주문 수가 숫자로 표시
  - [ ] 실패 주문이 0건이면 회색, 1건 이상이면 빨간색 아이콘
- [ ] **재고 충돌** 통계가 표시되는가?
  - [ ] 재고 충돌 수가 숫자로 표시
  - [ ] 재고 충돌이 0건이면 회색, 1건 이상이면 노란색 아이콘
- [ ] **"전체 보기"** 버튼이 표시되는가?
  - [ ] 클릭 시 `/dashboard/onewms`로 이동하는가?

### 경고 메시지
- [ ] 실패 주문 또는 재고 충돌이 있을 때 노란색 경고 박스 표시
- [ ] "⚠️ 주의가 필요한 항목이 있습니다" 메시지 표시

### 디자인 확인
- [ ] 카드 배경이 연한 파란색 (`bg-blue-50/50`)
- [ ] 카드 테두리가 파란색 (`border-blue-200`)
- [ ] 아이콘이 파란색 (`text-blue-600`)

## 3. 주문 상세 페이지 (`/orders/[id]`)

### ONEWMS 정보 카드 (OnewmsInfo)

#### 주문이 ONEWMS에 전송되지 않은 경우
- [ ] 회색 카드 표시
- [ ] "이 주문은 아직 ONEWMS에 전송되지 않았습니다" 메시지

#### 주문이 ONEWMS에 전송된 경우
- [ ] 연한 파란색 카드 표시 (`bg-blue-50/30`)
- [ ] **ONEWMS 주문번호** 표시
- [ ] **연동 상태** 배지 표시
  - [ ] 전송 대기: 회색
  - [ ] 전송 완료: 파란색
  - [ ] 출고 완료: 초록색
  - [ ] 전송 실패: 빨간색
  - [ ] 수동 처리 필요: 노란색

#### 송장번호가 있는 경우
- [ ] **송장번호** 표시
- [ ] **"이미지"** 버튼 표시
  - [ ] 클릭 시 GET `/api/onewms/delivery/invoice/[transNo]` 호출
  - [ ] 송장 이미지 새 탭에서 열림

#### 마지막 동기화 시간
- [ ] 마지막 동기화 시간이 한국 시간으로 표시

#### 상태 동기화 버튼
- [ ] **"상태 동기화"** 버튼 표시
  - [ ] 클릭 시 확인 대화상자 표시
  - [ ] 확인 시 POST `/api/onewms/delivery/update` 호출
  - [ ] 동기화 중 "동기화 중..." 표시 및 스피너
  - [ ] 완료 시 페이지 새로고침

#### CS 상태 경고
- [ ] CS 상태 또는 보류 상태가 0보다 크면 노란색 경고 박스 표시
- [ ] "⚠️ 고객 서비스 상태 확인 필요" 메시지
- [ ] CS 상태와 보류 상태 수치 표시

**테스트 주문**: ONEWMS에 전송된 주문 ID 사용

## 4. 재고 동기화 버튼 (StockSyncButton)

### 컴포넌트 독립 테스트
재고 동기화 버튼은 재사용 가능한 컴포넌트입니다.
상품 목록/상세 페이지가 구현되면 아래 항목들을 테스트하세요:

- [ ] 버튼이 표시되는가?
- [ ] 클릭 시 확인 대화상자 표시
  - [ ] 상품명이 있으면 확인 메시지에 상품명 포함
- [ ] 확인 시 POST `/api/onewms/stock/sync` 호출
  - [ ] `productId` 파라미터 포함
- [ ] 동기화 중 스피너 아이콘 표시
- [ ] 재고 충돌 감지 시 경고 메시지 표시
  - [ ] 플랫폼 재고, ONEWMS 재고, 차이 표시
- [ ] 동기화 성공 시 체크 아이콘 (3초간)
- [ ] 동기화 실패 시 X 아이콘 (3초간)

### Props 테스트
- [ ] `variant` prop으로 버튼 스타일 변경
- [ ] `size` prop으로 버튼 크기 변경
- [ ] `showLabel=false`로 아이콘만 표시
- [ ] `onSyncComplete` 콜백 호출 확인

## 5. API 엔드포인트 테스트

### 통계 API
```bash
curl http://localhost:3001/api/onewms/stats
```
- [ ] 200 OK 응답
- [ ] `orders.total`, `orders.failed`, `stock.conflicts` 데이터 포함

### 재고 동기화 API
```bash
curl -X POST http://localhost:3001/api/onewms/stock/sync \
  -H "Content-Type: application/json" \
  -d '{"productId": "prod_123"}'
```
- [ ] 200 OK 또는 401 Unauthorized (인증 필요)
- [ ] `syncedCount`, `conflicts` 데이터 포함

### 배송 상태 업데이트 API
```bash
curl -X POST http://localhost:3001/api/onewms/delivery/update \
  -H "Content-Type: application/json" \
  -d '{"orderId": "ord_123"}'
```
- [ ] 200 OK 또는 403 Forbidden (권한 필요)
- [ ] `updated`, `message` 데이터 포함

### 송장 이미지 API
```bash
curl http://localhost:3001/api/onewms/delivery/invoice/1234567890
```
- [ ] 200 OK 또는 404 Not Found
- [ ] `invoiceUrl` 데이터 포함
